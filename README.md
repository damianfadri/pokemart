# PokéMart
## Overview
PokéMart is a full-stack e-commerce web app for Pokémon-related items, built using a modern 2025 tech stack.

This project is part of an ongoing initiative to track the evolution of web development. By building applications like this, I get to learn the latest technologies firsthand and gain a clearer view of how tools and patterns evolve over time.

## Prerequisites
* npm

## Running the Project
1. Clone this project locally
2. Switch working directory to `src/web-app`.
```
cd src/web-app
```

3. Restore dependencies for frontend
```powershell
npm install
```

4. Run application
```powershell
npm run start
```
